小海豚博客

基于SpringBoot2.0 + Spring Data Jpa + Thymeleaf + Layui的个人博客

立即下载

当前版本：2020/2/29 更新日志

系统功能

1

博客模块

1.使用Layui作为前端样式，打造响应式，漂亮，简约的博客模板
2.使用EhCache作为缓存系统，常用的数据不用再次加载
3.使用JPA作为数据库框架，打造只要创建数据库，执行Jar包就可部署好网站

2

权限管理

使用Shiro权限框架，目前采用基于路径的权限，内置了扫描机制并加入资源库，管理员可以赋予角色，也可以把菜单资源设置为菜单。 并且内置了第三方QQ登录。

3

代码生成

代码生成功能可以帮助开发者快速开发项目，减少不必要的重复操作，让开发中花更多时间去注重业务， 让你二次开发更简单。

4

邮件系统

内置了邮件系统，可以设置留言时是否发送给自己或者被回复人，可以发送附件，HTML，普通的各种邮件。

技术栈

Spring Boot

Spring Data Jpa

Thymeleaf

Apache Shiro

EhCache

Layui

Jquery

Font Awesome
